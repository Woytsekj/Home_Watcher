# Home_Watcher
HomeWatcher is a mobile robot platform developed to allow users to remotely observe and interact with their environment from anywhere with an internet connection. It is designed to fill the gap between static cameras by providing a remote-controlled robot with real-time video feedback to a custom controller application. This was a Senior Design Project for the CEAS 2026 EXPO.

## What you will need 
### Hardware
* **Base Chassis:** SunFounder GalaxyRVR kit.
* **Communications:** An ESP32 WROVER-E module.
* **APP:** A Windows Laptop or an Andriod device.

### Cloud Stuff
* **ANY CLOUD VM** We used an AWS EC2
* **MQTT Broker:** Choose one of the many MQTT Brokers we used Mosquitto
* **STUN/TURN Server** Choose any STUN/TURN Server we used Coturn

## How to get the code Started
* **ESP32:** Was made for and compiled with PlatformIO
* **Arduino:** Was made for and complied with PlatformIO
* **PlatformIO:** If you have never used PlatformIO before, download the extension for VScode then download the whole file containing the platform.in and the open it in each respective file to open the whole project for each microcontroller
* **APP:** If you have an Android device you can download and use the APK. If you dont download the app code and compile it yourself
